【原创】 搭建网站遇到的问题
1 12月 2023
错误：npm: command not found： 在我搭建博客的时候。我已经安装了node.js和git bash，准备使用npm来安装Hexo。然而，当我在bash中输入npm install -g hexo-cli时，我却得到了一个错误提示：npm: command not found。 我感到非常困惑，因为我明明已经安装了node.js，npm应该是自带的。我开始在网上搜索这个问题，发现可能是因为npm没有被添加到环境变量中，导致bash找不到npm命令。
