JavaScript的经典例子（推荐）

作者：ZY 围观群众：0 更新于

1 12月 2023

JavaScript示例

1. Hello World：这是最基本的JavaScript代码，用于在控制台打印"Hello, World!"。这个例子展示了如何在控制台输出信息，这对于调试和学习JavaScript非常有用。
console.log("Hello, World!");

2. 函数（Function）：这个例子展示了如何定义和调用一个函数。函数是一段可以被重复使用的代码，可以提高代码的可读性和可维护性。
function greet(name) { return "Hello, " + name + "!"; } console.log(greet("World"));

3. 对象（Object）：这个例子展示了如何创建一个对象。在JavaScript中，对象是一个键值对的集合，可以用来存储和操作数据。
let car = { make: "Toyota", model: "Camry", year: 2020, displayCar: function() { return this.make + " " + this.model; } }; console.log(car.displayCar());

4. 数组（Array）：这个例子展示了如何创建和操作一个数组。数组是一种特殊的对象，用于存储有序的数据集合。
let fruits = ["Apple", "Banana", "Cherry"]; for(let i = 0; i < fruits.length; i++) { console.log(fruits[i]); }

5. Promise：这个例子展示了如何使用Promise进行异步编程。Promise是JavaScript中处理异步操作的一种方式，它可以帮助我们更好地组织和处理异步代码。
let promise = new Promise(function(resolve, reject) { setTimeout(function() { resolve("Promise resolved"); }, 1000); }); promise.then(function(value) { console.log(value); });

6. 箭头函数（Arrow Function）：这个例子展示了如何使用箭头函数。箭头函数是ES6中引入的新特性，它提供了一种更简洁的方式来写函数。
let square = x => x * x; console.log(square(5));

7. 模块（Module）：这个例子展示了如何使用模块来组织代码。模块可以帮助我们将代码分割成可重用的部分。
// math.js export function add(x, y) { return x + y; } // app.js import { add } from './math.js'; console.log(add(2, 3));

以上就是一些JavaScript的经典例子，它们展示了JavaScript的一些高级特性，如Promise、箭头函数和模块。这些例子对于理解和学习JavaScript非常有帮助。希望对你有所帮助！
